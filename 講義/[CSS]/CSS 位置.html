position 位置

用途

  • 將部分html的標籤顯示在最上層，會蓋住下層內容
  • 可依需求讓文字圖片等等不隨滑鼠滾輪移動
  • 例：我們瀏覽網頁常遇到的廣告，如下圖。

語法

  1. 設定要懸浮的標籤

    • position : static

      • 預設值，甚麼都不會變

    • position : fixed

      • 固定選擇到的標籤(包含該標籤底下的標籤)的位置，使其不隨滑鼠滾輪移動
      • 可設定固定在的位置(後面會說明如何設定位置)
      • 原本的位置會變成空的後面的標籤會補上來

    • position : relative

      • 相對自身位置移動
      • 會隨滑鼠滾輪移動
      • 搭配屬性(top、right、bottom、left)意義與其他position屬性的不同，可按F12觀察各方位position值與fixed的異同
        • 為設定距原位置該方位距離有多少
        • 例：下方程式碼會讓該容器向下位移100px
      

      position : relative;
top : 100px;

    • position : absolute

      • 屬性同於外層有position:fixed或relative或absolute的標籤
      • 移動限制在該外層容器內

    • position : sticky

      • 兼顧relative和fixed的性質
      • top、right、bottom、left至少設置其中一個，表示該方位到視窗的最小距離
      • 例：下方程式碼，當該容器距離視窗上方超過200px時不會有效果，但當該容器距離視窗上方要小於200px時會固定在原位置並保持與上方200px的距離
    

    position : sticky;
top : 200px;

  2. 調整位置位置

    • 觀察有設定position語法的標籤尺寸是不是都多了一個position呢? (按F12找看看)
    • 因為該標籤已經獨立了一層，所以position的值分別代表該標籤四個方位到邊界的距離
    • top : 設定值
      • 設定上position值(加上margin值為與上方邊框的距離)
    • right :　設定值
      • 設定右position值(加上margin值為與右方邊框的距離)
    • bottom : 設定值
      • 設定下position值(加上margin值為與下方邊框的距離)
    • left : 設定值
      • 設定左position值(加上margin值為與左方邊框的距離)
    • margin
      • 在margin的設定值中可以用auto代替，它會自動計算最大margin值並平分
      • 例1：下程式碼可以將該標籤設定為置中
      






      選擇器 {
    position : fixed;
    top : 0px;
    right : 0px;
    bottom : 0px;
    left : 0px;
    margin : auto;
      • 例2：下程式碼可以將該標籤設定為左右置中，但上下位置不變
      





      選擇器 {
    position : fixed;
    right : 0px;
    left : 0px;
    margin-left : auto;
    margin-right : auto;

問題集

為什麼設了position之後完全不會置中？

欸欸，為什麼我的廣告區塊用position置中之後不是我想像的樣子？

我明明已經把top, bottom, right, left都設定為0了，而且margin也設定成auto，到底為什麼？

請參考以下範例：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        *{
            padding: 0;
            margin: 0;
        }
        body, html{
            width: 100%;
            height: 100%;
        }
        .ad{
            background-color: #aaa;
            position: fixed;
            top: 0;
            bottom: 0;
            right: 0;
            left: 0;
            margin: auto;
        }
    </style>
</head>
<body>
    <div class="ad">
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Numquam, mollitia...
    </div>
</body>
</html>

你必須要給你的廣告區塊設定width跟height，否則：

  • 因為你設定了top, bottom, right, left為0，但是ad又沒有事先設定寬度，所以你的ad會自己往外擴張，變得與視窗上下左右的距離都是0。

  • 因為廣告區塊已經占滿了整個螢幕，所以你設定margin：auto沒有用，因為沒有剩餘的空間可以分配給margin。

下次不能置中的時候，不妨檢查一下你要設定的標籤有沒有設定寬高吧！
