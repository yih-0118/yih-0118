CSS放置位置

  1. 其他標籤屬性中

    在想要的改變樣式的標籤頭中加上屬性style

    例如：

    <body>
    <h1 style="css撰寫位置">
        這是h1標籤
    </h1>
</body>

  2. style標籤內

    在head標籤內建立一個style標籤

    <head>
    <style>
        /* css撰寫位置 */
    </style>
</head>

  3. 獨立css文件中

    1. 創立一個.css文件即可

    1. 再在要套用的html檔head標籤中使用link標籤
    <link rel="stylesheet" href=".css檔位置">
<!-- rel="stylesheet" 代表要連結到一個css檔 -->
<!-- href="" 雙引號中放css檔的相對或絕對位置' -->

CSS選擇器

CSS選擇器就是一個語法，讓你選擇要些修改的標籤

⚠️注意!
外層標籤的CSS也會套用到內層標籤

  • 寫在標籤裡的CSS就不用選擇器了

  • 通用選擇器：一次選擇全部標籤

    * {

}

    例如以下的CSS，會將所有標籤的padding、margin歸零。

    *{
    padding: 0;
    margin: 0;
}

  • 類型選擇器

    • 選擇特定標籤
    • 直接將標籤名稱打上就可以
    • 如果一次選擇多個標籤中間要加逗號，例如：
    h1,p {
    <!-- css寫在這裡-->
}

  • id選擇器

    • 在目標id名稱前面加上#
    #id名稱 {
    <!-- css寫在這裡-->
}

  • class選擇器

    • 在目標class名稱前面加上.
    .class名稱 {
    <!-- css寫在這裡-->
}

  • 後代選擇器

    • 選擇在特定標籤下的特定標籤
    • 標籤和標籤中以空白隔開
      例：
    div span {
    <!-- css寫在這裡-->
}
<!-- div的後代只要有span就會被選到-->

  • 子選擇器

    • 選擇在特定標籤下的特定標籤
    • 標籤和標籤中以>隔開
      例：
    div > span {
    <!-- css寫在這裡-->
}
<!-- 只看div的下一層標籤，更裡面的不會被選擇到-->

    ⚠️注意後代選擇器和子選擇器異同

    ❓想一想以上方後代選擇器和子選擇器給的範例，誰可以選到下方的span標籤?

    


    <div>
    <p> <span>哈囉哈囉</span> </p>
</div>
    點我看解答

    A: 後代選擇器。因為div和span中間隔一個p標籤，所以子選擇器不會選到span

  • 虛擬選擇器

    













    選擇器:link {
    /* css寫在這裡*/
}
/* 設定連結被點擊前樣式 */

選擇器:visited {
    /* css寫在這裡 */
}
/* 設定連結被點擊後的樣式 */

選擇器:hover {
    /* css寫在這裡 */
}
/* 設定滑鼠在標籤上方時的樣式 */
    • hover範例：
    













    <!DOCTYPE html>
<html>
    <head>
        <style>
            p:hover {
                color:red;
            }
        </style>
    </head>

    <body>
        <p>這是p標籤</p>
    </body>
</html>
