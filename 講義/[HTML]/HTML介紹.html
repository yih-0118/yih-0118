什麼是HTML

  • HTML（HyperText Markup Language，超文本標記語言）算是一種標記語言，與一般的程式語言不太一樣。主要是負責網頁的結構，像是文字、圖片及畫面排版等，可說是網頁的核心。

要怎麼看到網頁上的HTML

  • 滑鼠右鍵後，
    點選「檢視網頁原始碼」/「檢視網頁來源」

基本的HTML架構

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8"/>
    <title>My first page</title>
</head>
<body>
    <h1>Hello, world!</h1>
</body>
</html>

HTML元素的基本觀念

  • 標籤的基本格式
<tagname attributes=""></tagname>
<tagname attributes=""/>
  • tagname為標籤名稱。
  • attributes為屬性，大部分標籤具有不同屬性，但也有共通的共同屬性。
  • <tagname attributes="">為opening tag；
  • </tagname>為closing tag。
  • 標籤的巢狀結構
  • 標籤的共同屬性、個別的屬性

超連結 (Anchor)

  • href (Hypertext reference) 後面的雙引號中放入所要連結的東西，可以是網頁（絕對位置或相對位置）、網頁的特定段落、信箱甚至是電話。
  • target：決定連結打開的方式
    • _self 在原本的頁面開啟
    • _blank 在新視窗開啟

表單(form)

表單的組成

表單主要由兩個元素所組成，分別是：

  • input：透過控制type，input可以有很多不同類型，包含文字輸入框、選項、下拉式清單等

  • label：label則是用來顯示在螢幕上的文字，跟相對應的input綁定在一起。以底下的範例來說，"電話"這個label的for屬性必須跟input的id一樣。

  • Example:
<form action="/action_page.php">
  <label for="phone">電話：</label><br/>
  <input type="text" id="phone" name="phone" value=""/><br/>
  <label for="name">姓名：</label><br/>
  <input type="text" id="name" name="name" value=""/><br/><br/>
  <input type="submit" value="Submit"/>
</form>

input的種類

1. type="checkbox"

  • 描述：多選框，用戶可以選擇多個選項。
  • 用法：
<input type="checkbox" name="vehicle" value="Bike"> 
ubike
<input type="checkbox" name="vehicle" value="Car"> C300
  • 展示：
    騎自行車 開汽車

2. type="radio"

  • 描述：單選框，用戶只能選擇一個選項。
  • 用法：
<input type="radio" name="gender" value="male"> 男
<input type="radio" name="gender" value="female"> 女
  • 展示：

3. type="file"

  • 描述：用戶可以選擇上傳文件。
  • 用法：
<input type="file" name="fileToUpload" id="fileToUpload">
  • 展示：
  • 注意事項：需要使用 <form> 元素包裹以便提交文件。

4. type="hidden"

  • 描述：隱藏字段，不可見於用戶，用於在表單中傳遞隱藏數據。
  • 用法：
<input type="hidden" name="sessionID" value="12345">`

5. type="text"

  • 描述：單行文本輸入框。
  • 用法：
<input type="text" name="username" placeholder="請輸入您的用戶名">
  • 展示：

6. type="password"

  • 描述：密碼輸入框，文本將被遮蔽。
  • 用法：
<input type="password" name="password" placeholder="請輸入密碼">
  • 展示：

7. type="image"

  • 描述：圖像按鈕，當用戶點擊時觸發表單提交。
  • 用法：
<input type="image" src="submit_button.png" alt="提交">

8. type="reset"

  • 描述：重置按鈕，用戶點擊後重置表單中的所有字段。
  • 用法：
<input type="reset" value="重置">`
  • 展示：

9. type="button"

  • 描述：普通按鈕，通常用於觸發JavaScript函數。
  • 用法：
<input type="button" value="點擊我" onclick="myFunction()">
  • 展示：

10. type="submit"

  • 描述：提交按鈕，用戶點擊後提交表單。
  • 用法：
<input type="submit" value="提交">
  • 展示：

11. type="color"

  • 描述：顏色選擇器，用戶可以選擇顏色。

  • 用法：

    html Copy code

    <input type="color" name="color" value="#ff0000">

12. type="date"

  • 描述：日期選擇器，用戶可以選擇日期。
  • 用法：
<input type="date" name="birthdate">
  • 展示：

13. type="time"

  • 描述：時間選擇器，用戶可以選擇時間。
  • 用法：
<input type="time" name="meeting-time">
  • 展示：

14. type="month"

  • 描述：月份選擇器，用戶可以選擇年份和月份。
  • 用法：
<input type="month" name="start-month">
  • 展示：

15. type="datetime-local"

  • 描述：日期和時間選擇器，用戶可以選擇日期和時間。
  • 用法：
<input type="datetime-local" name="meeting-datetime">
  • 展示：

16. type="week"

  • 描述：周數選擇器，用戶可以選擇年份和周數。
  • 用法：
<input type="week" name="event-week">
  • 展示：

17. type="number"

  • 描述：數字輸入框，只允許輸入數字。
  • 用法：
<input type="number" name="quantity" min="1" max="100" step="1">
  • 展示：

18. type="search"

  • 描述：搜索框，用戶可以輸入搜索關鍵詞。
  • 用法：
<input type="search" name="search" placeholder="搜尋...">
  • 展示：

19. type="range"

  • 描述：滑塊輸入框，用戶可以選擇一個範圍內的數值。
  • 用法：
<input type="range" name="volume" min="0" max="100" step="1">
  • 展示：

內嵌框架

<iframe> 標籤基本結構

<iframe src="URL" width="width" height="height" frameborder="0"></iframe>
  • src 屬性：指定要嵌入的網頁的URL。
  • width 屬性：指定 <iframe> 的寬度（像素或百分比）。
  • height 屬性：指定 <iframe> 的高度（像素或百分比）。
  • frameborder 屬性：設置為 “0” 可以移除 <iframe> 周圍的邊框。
